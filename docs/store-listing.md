## ENGLISH

### Short Description (65 chars — max 80)
Scientific calculator — trig, statistics, astronomy & electronics

### Full Description
Scientific Calculator Pro is a powerful, multi-domain scientific calculator designed for students, engineers, and scientists.

CALCULATOR
• Trigonometric functions (sin, cos, tan) in degrees or radians
• Inverse trigonometric functions (sin⁻¹, cos⁻¹, tan⁻¹)
• Logarithm (log, ln) and exponential functions (eˣ, 10ˣ)
• Factorial (n!), permutations (nPr) and combinations (nCr)
• Power (xʸ), nth root (ʸ√x), square root, reciprocal (1/x)
• Memory functions (M+, M−, MR, MC)
• Casio-inspired dot matrix LCD display

ASTRONOMY MODULE
• Gravitational acceleration on any celestial body
• Schwarzschild radius — black hole event horizon
• Escape velocity
• Hubble's Law — recessional velocity and distance
• Kepler's Third Law — orbital period

ELECTRONICS MODULE
• Ohm's Law (V = IR)
• Electrical power (P = VI)
• RC circuit — time constant, charge/discharge curves
• LC resonance frequency
• Voltage divider

UNIT CONVERTER
• Length, mass, temperature, pressure and more

DESIGN
Elegant dark interface with a Casio ClassWiz-inspired dot matrix LCD display. All calculations run entirely on-device — no internet connection required for computations.

PRIVACY
This application does not collect, store, or transmit any personal data.
